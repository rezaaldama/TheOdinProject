# Project: Library

A simple web application to manage a personal book collection. This project focuses on practicing JavaScript Objects, Constructors, and DOM manipulation.

## Features
- Add new books to the library (Title, Author, Pages, Read Status).
- View all books in a responsive grid/table layout.
- Toggle the "Read" status of any book.
- Delete books from the collection.
- Persistent UI updates using JavaScript.

## Technologies Used
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (Plain Vanilla)

## Reflections
- Practiced using `Object Constructors` and `Prototypes`.
- Learned how to manipulate the DOM to dynamically display objects from an array.
- Implemented form validation and modal logic for adding new entries.
